Aula 2: Links e Imagens - Conectando e Ilustrando a Web

Após aprender a estrutura básica do HTML, é hora de dar vida às suas páginas! Nesta aula, você descobrirá como adicionar **links** para navegar entre conteúdos e como inserir **imagens** para tornar seu site visualmente mais atraente e informativo.

1. Links (Hyperlinks) - A Conexão da Internet

Os links são o coração da World Wide Web, permitindo que os usuários saltem de um documento para outro. No HTML, usamos a tag <a> (âncora) para criar um link.

Estrutura Básica do Link:

A tag <a> precisa do atributo href (Hypertext REFerence), que define o destino do link.

<a href="destino_do_link.html">Texto que o usuário vai clicar</a>
  • href: O endereço (URL) para onde o link aponta.
  • Texto que o usuário vai clicar: O texto visível que o usuário interagirá.

Tipos de Links:

Links Internos (dentro do seu projeto):

Apontam para outros arquivos HTML na sua pasta ou em subpastas. O caminho é relativo à posição do arquivo atual.


<!-- Link para um arquivo na mesma pasta -->
<a href="aula1.html">Ir para a Aula 1</a>

<!-- Link para um arquivo dentro da pasta 'paginas' -->
<a href="paginas/sobre.html">Nossa História</a>

<!-- Link para um arquivo um nível acima (usando '../') -->
<a href="../index.html">Voltar para a Home</a>

Links Externos (para outros sites):

Apontam para qualquer endereço na internet. Sempre use o protocolo completo (http:// ou https://).

<a href="https://www.google.com">Pesquisar no Google</a>

Abrindo Links em Novas Abas (target="_blank"):

Para que o link seja aberto em uma nova aba do navegador (e o usuário não saia da sua página atual), use o atributo target="_blank".

<a href="https://developer.mozilla.org/pt-BR/" target="_blank">Documentação MDN</a>

Links para Seções Específicas (Âncoras - #id):

Você pode criar um "salto" para uma parte específica da mesma página (ou de outra página) usando o atributo id. Primeiro, dê um id único ao elemento de destino, depois crie um link para ele com #nome_do_id.


<!-- Na seção de destino -->
<section id="topo-pagina">
    <h2>Início da Página</h2>
</section>

<!-- O link para a âncora -->
<a href="#topo-pagina">Ir para o Topo</a>

Links Especiais (E-mail e Telefone):

Você pode configurar links para iniciar um e-mail ou uma ligação telefônica:


<a href="mailto:contato@exemplo.com">Enviar E-mail</a>
<a href="tel:+5511987654321">Ligar Agora</a>

2. Imagens - Ilustrando seu Conteúdo

As imagens são essenciais para aprimorar o design e a clareza da sua página. A tag para inserir uma imagem é a <img>, que é uma tag vazia (não tem fechamento).

Estrutura Básica da Imagem:

A tag <img> precisa de dois atributos cruciais: src e alt.

<img src="caminho_da_sua_imagem.jpg" alt="Descrição útil da imagem">
  • src (source): O caminho para o arquivo da imagem. Pode ser um caminho local (relativo) ou um URL (absoluto).
  • alt (alternative text): Texto alternativo que é exibido se a imagem não carregar, ou lido por leitores de tela para acessibilidade. **É fundamental para SEO e inclusão!**

Exemplos de src:

Imagem Local (na pasta imagens/):

Se você tem uma imagem minha-foto.jpg dentro da sua pasta imagens/, o caminho será:

<img src="imagens/minha-foto.jpg" alt="Foto de perfil de [Seu Nome]">

Imagem de um URL Externo:

Você pode usar imagens hospedadas em outros sites, mas isso pode gerar problemas se a imagem for removida do site original.

<img src="https://picsum.photos/400/300" alt="Imagem aleatória de paisagem">

Atributos de Dimensão (width e height):

Embora seja possível definir a largura e altura da imagem diretamente no HTML, é uma prática melhor usar o CSS para isso, pois oferece mais controle e flexibilidade para a responsividade (que veremos em aulas futuras).

<img src="imagens/logo.png" alt="Logotipo da empresa" width="150" height="auto">
