Aula 6: Modelo de Caixa - Como os Elementos Ocupam Espaço

No CSS, todo elemento HTML é tratado como uma caixa retangular. Entender o Modelo de Caixa (Box Model) é fundamental para controlar o espaçamento, o tamanho e o posicionamento dos elementos na sua página. Ele é composto por quatro partes principais: Conteúdo, Preenchimento (Padding), Borda (Border) e Margem (Margin).

Anatomia do Modelo de Caixa

Imagine que cada elemento HTML é como uma caixa com várias camadas, como uma cebola. Do centro para fora, temos:

Conteúdo
Padding
Border
Margin

1. Conteúdo (Content)

  • É a área central da caixa, onde o conteúdo real do elemento é exibido. Pode ser texto, imagens, vídeos, etc.
  • Seu tamanho é controlado pelas propriedades width (largura) e height (altura).

p {
    width: 200px; /* Largura do conteúdo */
    height: 100px; /* Altura do conteúdo */
    background-color: lightblue;
}

2. Preenchimento (Padding)

  • O espaço entre o conteúdo e a borda do elemento.
  • É usado para dar "respiro" ao conteúdo, evitando que ele fique colado nas bordas.
  • O padding recebe a mesma cor de fundo do elemento.
  • Pode ser aplicado uniformemente ou em lados específicos (padding-top, padding-right, padding-bottom, padding-left).

.caixa {
    padding: 20px; /* 20px em todos os lados */
    padding-top: 10px; /* 10px em cima */
    padding-right: 30px; /* 30px à direita */
    /* Shorthand: top right bottom left */
    padding: 10px 20px 30px 40px;
    /* Shorthand: top/bottom left/right */
    padding: 15px 30px;
}

3. Borda (Border)

  • A linha que envolve o padding e o conteúdo.
  • Serve para separar visualmente o elemento de outros.
  • Três propriedades principais para borda: border-width (espessura), border-style (tipo da linha, ex: solid, dashed, dotted) e border-color (cor).
  • Assim como o padding, pode ser definida para lados específicos.

.caixa {
    border-width: 2px;
    border-style: solid;
    border-color: black;
    /* Shorthand: width style color */
    border: 2px solid #ccc;
    /* Borda apenas na parte inferior */
    border-bottom: 1px dashed red;
}

4. Margem (Margin)

  • O espaço externo do elemento, entre a borda do elemento e os elementos vizinhos.
  • Usado para criar distância entre um elemento e outro.
  • O margin é sempre transparente.
  • Assim como o padding, pode ser aplicado uniformemente ou em lados específicos (margin-top, margin-right, margin-bottom, margin-left).
  • Colapso de Margens: Margens verticais adjacentes (entre elementos acima/abaixo) se "colapsam", ou seja, a maior margem prevalece, não somam. Margens horizontais (entre elementos lado a lado) se somam normalmente.

.caixa {
    margin: 30px; /* 30px em todos os lados */
    margin-top: 15px; /* 15px em cima */
    margin-left: auto; /* Centraliza elementos de bloco horizontalmente (com width definido) */
    margin-right: auto;
    /* Shorthand: top right bottom left */
    margin: 5px 10px 15px 20px;
}

Centralizando Elementos de Bloco: Para centralizar um elemento de bloco horizontalmente, defina uma largura (`width`) para ele e use `margin: auto;` (ou `margin-left: auto; margin-right: auto;`).


.elemento-centralizado {
    width: 60%; /* Defina uma largura */
    margin: 0 auto; /* 0px de margem vertical, 'auto' nas laterais */
    background-color: #f0f8ff;
    padding: 20px;
    border: 1px solid #add8e6;
}

Propriedade box-sizing:

Por padrão, `width` e `height` definem o tamanho da **área de conteúdo**. Isso significa que `padding` e `border` são *adicionados* a essa largura/altura, aumentando o tamanho total do elemento.

Com box-sizing: border-box;, width e height incluem o padding e a borda. Isso simplifica muito o cálculo de layouts, pois o tamanho total do elemento é exatamente o que você define. **É uma prática recomendada!**


* { /* Aplica-se a todos os elementos */
    box-sizing: border-box;
}

.minha-caixa {
    width: 200px; /* 200px incluindo padding e borda */
    padding: 20px;
    border: 5px solid black;
    /* Conteúdo terá 200 - (20+20) - (5+5) = 150px de largura */
}

HTML:


<div class="card-central">
    <h3>Título do Card</h3>
    <p>Este é o conteúdo do card.</p>
</div>

Problema 3: Espaçamento entre Itens de Lista

Você tem uma lista não ordenada de itens e quer que cada item tenha 10px de espaço na parte inferior para separá-los visualmente, mas sem adicionar espaço extra ao último item da lista.

HTML:


<ul class="lista-itens">
    <li>Item Um</li>
    <li>Item Dois</li>
    <li>Item Três</li>
</ul>
