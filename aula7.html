Aula 7: Posicionamento Básico - Controlando Onde os Elementos Ficam

Após entender o Modelo de Caixa, o próximo passo é dominar como os elementos são posicionados na página. A propriedade position no CSS é a chave para isso, permitindo que você tire elementos do seu fluxo normal e os coloque exatamente onde deseja.

Tipos de Posicionamento (position)

Existem cinco valores principais para a propriedade position, cada um com um comportamento distinto:

1. position: static; (Padrão)

  • É o valor **padrão** para todos os elementos HTML.
  • Os elementos são renderizados na ordem em que aparecem no fluxo normal do documento (da esquerda para a direita, de cima para baixo).
  • As propriedades top, right, bottom, left e z-index **não têm efeito** em elementos com position: static;.

.elemento-normal {
    position: static; /* Não faz diferença, é o padrão */
}

2. position: relative; (Relativo à Posição Normal)

  • O elemento é posicionado **relativo à sua posição normal** no fluxo do documento.
  • Você pode usar top, right, bottom, left para deslocá-lo.
  • Importante: Mesmo deslocado, o espaço original que o elemento ocuparia no fluxo **ainda é reservado**, o que pode causar "buracos" ou sobreposições.

.elemento-relativo {
    position: relative;
    top: 20px;   /* Move 20px para baixo da posição original */
    left: 30px;  /* Move 30px para a direita da posição original */
}

3. position: absolute; (Absoluto ao Pai Posicionado)

  • O elemento é posicionado **absolutamente em relação ao seu ancestral posicionado mais próximo** (ou seja, um pai que tenha position: relative;, absolute;, fixed; ou sticky;).
  • Se não houver nenhum ancestral posicionado, ele se posiciona em relação ao elemento <body>.
  • **É removido do fluxo normal** do documento. O espaço que ele ocupava é liberado, e outros elementos se comportam como se ele não existisse ali.
  • Use top, right, bottom, left para definir sua posição exata.

/* O elemento pai deve ter position: relative; */
.pai-posicionado {
    position: relative;
}

.elemento-absoluto {
    position: absolute;
    top: 0;      /* 0px do topo do pai */
    right: 0;    /* 0px da direita do pai */
}

4. position: fixed; (Fixo na Tela)

  • O elemento é posicionado **absolutamente em relação à janela de visualização (viewport)** do navegador.
  • **Permanece fixo na mesma posição** mesmo quando a página é rolada.
  • Também é removido do fluxo normal do documento.
  • Comum para cabeçalhos fixos, rodapés, botões "voltar ao topo" ou menus de navegação flutuantes.

.barra-fixa {
    position: fixed;
    top: 0;      /* Fica fixo no topo da tela */
    width: 100%; /* Ocupa a largura total */
    background-color: #333;
    color: white;
    padding: 10px;
    z-index: 999; /* Garante que fique acima de outros elementos */
}

5. position: sticky; (Fixo com Rolagem)

  • Um híbrido de relative e fixed.
  • O elemento se comporta como relative até que uma certa posição de rolagem seja atingida, então ele "gruda" na tela como um elemento fixed.
  • Precisa de uma propriedade top, right, bottom ou left para funcionar.
  • Muito útil para cabeçalhos de seções, barras de navegação que aparecem ao rolar, etc.

.menu-sticky {
    position: sticky;
    top: 0; /* Cola no topo da tela quando o elemento o alcança */
    background-color: #f8f8f8;
    padding: 10px;
    border-bottom: 1px solid #ddd;
}

A propriedade z-index

  • Controla a ordem de empilhamento de elementos posicionados (não `static`).
  • Elementos com maior z-index ficam **acima** de elementos com menor z-index.
  • Só funciona em elementos que têm position diferente de `static`.
  • Valores podem ser positivos, negativos ou zero.

.caixa-acima {
    position: absolute;
    z-index: 2; /* Fica acima da caixa-abaixo */
}
.caixa-abaixo {
    position: absolute;
    z-index: 1;
}

Problema 3: Elemento "Sticky" em uma Seção

Crie uma seção longa de conteúdo. Dentro dela, adicione um título secundário (`<h3>`) que, ao rolar a página, "grude" no topo da viewport quando ele chega lá, agindo como um subtítulo fixo para aquela seção enquanto o restante do conteúdo rola.

HTML:


<section class="conteudo-longo">
    <p>Texto antes do sticky...</p>
    <h3 class="sticky-title">Seção de Detalhes</h3>
    <p>Conteúdo da seção de detalhes que rola abaixo do título...</p>
</section>

CSS:


/* Insira seu código CSS aqui para resolver o Problema 3 */
.conteudo-longo {
    height: 600px; /* Apenas para criar rolagem de exemplo */
    overflow-y: scroll;
    border: 1px solid #ddd;
    padding: 15px;
}
.sticky-title {
    /* ... */
}
