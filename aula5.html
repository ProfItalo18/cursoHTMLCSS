Aula 5: Introdução ao CSS - Dê Estilo às Suas Páginas!

Até agora, suas páginas HTML parecem um "esqueleto" sem muita graça. É hora de vesti-las com roupas bonitas! Nesta aula, você será introduzido ao CSS (Cascading Style Sheets), a linguagem responsável por todo o visual da sua página: cores, fontes, tamanhos, espaçamentos, posicionamento e muito mais.

O Que é CSS e Por Que Usá-lo?

CSS é a linguagem de estilo que descreve a apresentação de um documento escrito em HTML. Pense no HTML como a estrutura da casa e o CSS como a decoração: a cor das paredes, o tipo de piso, o tamanho das janelas.

Usamos CSS para:

  • Controlar cores de texto, fundo e bordas.
  • Definir tipos, tamanhos e estilos de fonte.
  • Organizar o layout dos elementos na página (colunas, alinhamentos).
  • Adicionar animações e transições.
  • Tornar a página responsiva (adaptável a diferentes tamanhos de tela).

A principal vantagem é separar o conteúdo (HTML) da apresentação (CSS), tornando seu código mais organizado, fácil de manter e reutilizável.

Sintaxe Básica do CSS: Regras e Declarações

Uma regra CSS consiste em um seletor e um bloco de declarações.


seletor {
    propriedade: valor;
    propriedade-dois: valor-dois;
    /* ... outras declarações ... */
}
  • Seletor: Aponta para o elemento HTML que você quer estilizar (ex: `p`, `h1`, `.minha-classe`, `#meu-id`).
  • Declaração: Uma ou mais declarações separadas por ponto e vírgula (`;`). Cada declaração é um par de propriedade e valor, separados por dois pontos (`:`).
    • Propriedade: O atributo de estilo que você quer mudar (ex: `color`, `font-size`, `background-color`).
    • Valor: O valor que você quer atribuir à propriedade (ex: `blue`, `16px`, `red`).

Exemplo: Estilizando todos os parágrafos para serem azuis.


p {
    color: blue; /* Define a cor do texto como azul */
    font-size: 18px; /* Define o tamanho da fonte como 18 pixels */
}

Este parágrafo é um exemplo de como o CSS funciona.

Onde Colocar o CSS: 3 Maneiras de Incluir Estilos

Existem três formas principais de aplicar CSS a uma página HTML:

1. CSS Externo (Recomendado!)

É a forma mais comum e recomendada. Você cria um arquivo .css separado (ex: style.css) e o vincula ao seu HTML usando a tag <link> dentro do <head>.

No seu arquivo HTML (`.html`):


<head>
    <link rel="stylesheet" href="style.css">
</head>

No seu arquivo CSS (`style.css`):


/* style.css */
body {
    font-family: Arial, sans-serif;
    background-color: #f0f0f0;
}

Vantagens: Código organizado, fácil de manter, estilos reutilizáveis em várias páginas.

2. CSS Interno (<style> Tag)

Você escreve o CSS diretamente dentro da tag <style>, que fica no <head> do seu documento HTML.

No seu arquivo HTML (`.html`):


<head>
    <style>
        h1 {
            color: purple;
        }
    </style>
</head>
<body>
    <h1>Este título é roxo!</h1>
</body>

Este título é roxo! (Exemplo Interno)

Vantagens: Útil para estilos específicos de uma única página. **Desvantagens:** Mistura conteúdo e estilo, difícil de manter em projetos maiores.

3. CSS Inline (Atributo style)

Você aplica estilos diretamente a um elemento HTML usando o atributo style na própria tag.

No seu arquivo HTML (`.html`):


<p style="color: green; background-color: lightgreen;">Este parágrafo tem estilo inline.</p>

Este parágrafo tem estilo inline. (Exemplo Inline)

Vantagens: Para estilos muito pontuais e específicos. **Desvantagens:** Pior forma para organização, não reutilizável, difícil de manter, anula outros estilos.

Seletores CSS Básicos: Quem você quer estilizar?

Seletores são como você "seleciona" os elementos HTML para aplicar os estilos CSS.

1. Seletor de Tipo (Tag)

Seleciona todos os elementos de um determinado tipo (tag).


/* Estiliza todos os parágrafos */
p {
    font-family: sans-serif;
}

/* Estiliza todos os h2 */
h2 {
    text-align: center;
}

2. Seletor de Classe (.classe)

Seleciona elementos que possuem um atributo class específico. Você pode aplicar a mesma classe a vários elementos.

No HTML: <div class="meu-componente">...</div>

No CSS:


/* Estiliza elementos com a classe 'meu-componente' */
.meu-componente {
    border: 1px solid blue;
    padding: 10px;
}
Este é um elemento com a classe 'exemplo-classe'.
Outro elemento com a mesma classe, mas com outro estilo inline para diferenciação.

3. Seletor de ID (#id)

Seleciona um **único** elemento com um atributo id específico. O id deve ser único na página.

No HTML: <p id="meu-paragrafo-unico">...</p>

No CSS:


/* Estiliza o elemento com o ID 'meu-paragrafo-unico' */
#meu-paragrafo-unico {
    color: red;
    font-weight: bold;
}

Este parágrafo tem um ID único e um estilo específico.

Precedência: Estilos inline > Estilos ID > Estilos Classe > Estilos de Tipo. (Veremos mais sobre isso em especificidade CSS).

Primeiras Propriedades CSS Essenciais:

  • color: Cor do texto (ex: `color: red;`, `color: #FF0000;`, `color: rgb(255, 0, 0);`).
  • background-color: Cor de fundo de um elemento.
  • font-size: Tamanho da fonte (ex: `16px`, `1.2em`, `150%`).
  • font-family: Tipo da fonte (ex: `Arial, sans-serif;`).
  • text-align: Alinhamento do texto (ex: `left`, `center`, `right`, `justify`).
  • border: Adiciona uma borda (ex: `border: 1px solid black;`).
  • padding: Espaço interno entre o conteúdo e a borda do elemento.
  • margin: Espaço externo entre a borda do elemento e outros elementos.

<!-- HTML -->
<p>Este é o primeiro parágrafo.</p>
<p>Este é o segundo parágrafo.</p>
<p>Este é o terceiro parágrafo.</p>

<!-- CSS -->

<!-- HTML -->
<div class="card-destaque">
    <h3>Oferta Especial!</h3>
    <p>Aproveite nossos descontos incríveis.</p>
</div>

<!-- CSS -->

<!-- HTML -->
