Aula 11: Transições e Animações CSS - Dando Vida aos Elementos

Transições e animações CSS permitem criar efeitos visuais fluidos e atraentes sem a necessidade de JavaScript complexo. Elas são fundamentais para melhorar a experiência do usuário, indicando mudanças de estado e tornando a interface mais dinâmica.

Transições CSS: Mudanças Suaves de Estado

Transições são usadas para criar um efeito suave e gradual quando uma propriedade CSS de um elemento muda de um estado para outro (ex: no `hover`, `focus`, ou quando uma classe é adicionada/removida via JavaScript).

Propriedades das Transições:

  • transition-property: Qual propriedade CSS será transicionada (ex: `background-color`, `opacity`, `transform`, `all`). Use `all` para transicionar todas as propriedades que suportam transição.
  • transition-duration: A duração da transição (ex: `0.5s`, `200ms`).
  • transition-timing-function: A velocidade da transição (como ela acelera ou desacelera).
    • ease (padrão): Inicia e termina lentamente.
    • linear: Velocidade constante.
    • ease-in: Inicia lentamente.
    • ease-out: Termina lentamente.
    • ease-in-out: Inicia e termina lentamente (mais suave que `ease`).
    • cubic-bezier(...): Curva de velocidade personalizada.
  • transition-delay: Um atraso antes do início da transição.

O shorthand transition combina todas essas propriedades na ordem: `property duration timing-function delay`.


.minha-caixa {
    background-color: blue;
    transition: background-color 0.5s ease-in-out; /* Transiciona a cor de fundo em 0.5s */
}

.minha-caixa:hover {
    background-color: red;
}

/* Exemplo com múltiplas propriedades */
.meu-botao {
    transform: scale(1);
    opacity: 1;
    transition: transform 0.3s ease-out, opacity 0.5s linear;
}

.meu-botao:active {
    transform: scale(0.95);
    opacity: 0.8;
}

Animações CSS: Sequências de Estilos Complexas

Animações são mais poderosas que transições porque permitem definir múltiplos "quadros-chave" (keyframes) para um efeito que acontece automaticamente, sem a necessidade de um evento de mouse ou clique. Elas podem ser repetidas, atrasadas, revertidas, etc.

Passos para Criar uma Animação:

  1. Definir os Keyframes: Use a regra @keyframes para criar uma sequência de estilos. Você define os estilos em porcentagens (0%, 50%, 100%) ou com from (0%) e to (100%). 
    
@keyframes nomeDaAnimacao {
    0% {
        transform: translateX(0);
    }
    50% {
        transform: translateX(100px);
    }
    100% {
        transform: translateX(0);
    }
}
  2. Aplicar a Animação ao Elemento: Use as propriedades `animation` para vincular os keyframes a um elemento.
    • animation-name: O nome da animação (`nomeDaAnimacao`).
    • animation-duration: Duração de um ciclo da animação.
    • animation-timing-function: Velocidade (igual às transições).
    • animation-delay: Atraso antes do início.
    • animation-iteration-count: Quantas vezes a animação deve rodar (número ou `infinite`).
    • animation-direction: Se a animação deve ir e voltar (`alternate`, `alternate-reverse`).
    • animation-fill-mode: Como os estilos do elemento se comportam antes e depois da animação (forwards, backwards, both, `none`). `forwards` mantém o estado final da animação.
    • animation-play-state: `running` ou `paused`.

O shorthand animation combina todas essas propriedades na ordem: `name duration timing-function delay iteration-count direction fill-mode play-state`.


.meu-elemento-animado {
    animation-name: nomeDaAnimacao;
    animation-duration: 2s;
    animation-iteration-count: infinite; /* Roda infinitamente */
    animation-direction: alternate; /* Vai e volta */
    animation-fill-mode: forwards; /* Mantém o estado final */
    /* Shorthand: */
    /* animation: nomeDaAnimacao 2s infinite alternate forwards; */
}

Problema 4: Barra de Progresso Carregando

Crie uma barra de progresso. Use uma animação para simular o "carregamento", onde uma barra interna se expande de 0% a 100% da largura da barra externa, e então se reinicia.

HTML:


<div class="progress-bar-container">
    <div class="progress-bar-fill"></div>
</div>

CSS:


/* Insira seu código CSS aqui para resolver o Problema 4 */
.progress-bar-container {
    width: 80%;
    height: 20px;
    background-color: #e0e0e0;
    border-radius: 10px;
    overflow: hidden; /* Garante que a barra interna não vaze */
    margin: 30px auto;
}

.progress-bar-fill {
    height: 100%;
    width: 0%; /* Inicia em 0 */
    background-color: #2196F3;
    border-radius: 10px;
    /* Defina sua animação aqui */
}

/* Defina seus @keyframes aqui */
@keyframes loadProgress {
    /* ... */
}
