Aula 3: Listas e Tabelas - Organize seu Conteúdo com Estrutura

Nesta aula, você aprenderá a organizar informações de forma clara e estruturada usando listas e tabelas em HTML. Esses elementos são fundamentais para apresentar dados de maneira legível e acessível.

1. Listas HTML: Organizando Itens

As listas são perfeitas para agrupar itens relacionados. Existem três tipos principais de listas em HTML:

1.1. Listas Não Ordenadas (<ul> - Unordered List)

Usadas para itens onde a ordem não importa (por exemplo, uma lista de compras, habilidades). Os itens são marcados por símbolos (geralmente bolinhas).


<ul>
    <li>Maçãs</li>
    <li>Bananas</li>
    <li>Laranjas</li>
</ul>

Resultado:

  • Maçãs
  • Bananas
  • Laranjas

1.2. Listas Ordenadas (<ol> - Ordered List)

Usadas para itens que possuem uma ordem específica (por exemplo, um passo a passo de uma receita, rankings). Os itens são marcados por números ou letras.


<ol>
    <li>Ferva a água</li>
    <li>Adicione o macarrão</li>
    <li>Cozinhe por 8 minutos</li>
</ol>

Resultado:

  1. Ferva a água
  2. Adicione o macarrão
  3. Cozinhe por 8 minutos

Você pode mudar o tipo de marcador usando o atributo type no <ol> (ex: type="a" para letras, type="I" para números romanos maiúsculos). Mas é mais comum e recomendado estilizar com CSS.

1.3. Listas de Definição (<dl> - Description List)

Usadas para listar termos e suas descrições (por exemplo, um glossário). Elas não usam <li>, mas sim:

  • <dt> (Definition Term): O termo a ser definido.
  • <dd> (Definition Description): A descrição do termo.

<dl>
    <dt>HTML</dt>
    <dd>Linguagem de Marcação de Hipertexto.</dd>

    <dt>CSS</dt>
    <dd>Folhas de Estilo em Cascata.</dd>
</dl>

Resultado:

HTML
Linguagem de Marcação de Hipertexto.
CSS
Folhas de Estilo em Cascata.

2. Tabelas HTML: Exibindo Dados Tabulares

As tabelas são ideais para exibir dados em formato de grade (linhas e colunas), como informações financeiras, horários, ou listas de produtos com atributos.

Estrutura Básica de uma Tabela:


<table class="aula-table">
    <thead>
        <tr>
            <th>Cabeçalho 1</th>
            <th>Cabeçalho 2</th>
            <th>Cabeçalho 3</th>
        </tr>
    </thead>
    <tbody>
        <tr>
            <td>Dado A1</td>
            <td>Dado A2</td>
            <td>Dado A3</td>
        </tr>
        <tr>
            <td>Dado B1</td>
            <td>Dado B2</td>
            <td>Dado B3</td>
        </tr>
    </tbody>
    <tfoot>
        <tr>
            <td colspan="3">Rodapé da tabela</td>
        </tr>
    </tfoot>
</table>

Resultado (com o CSS já aplicado):

Cabeçalho 1 Cabeçalho 2 Cabeçalho 3
Dado A1 Dado A2 Dado A3
Dado B1 Dado B2 Dado B3
Rodapé da tabela

Detalhando as Tags da Tabela:

  • <table>: Contém todo o conteúdo da tabela.
  • <thead>: Grupo de cabeçalho da tabela. Contém as informações do cabeçalho.
    • <tr> (Table Row): Define uma linha na tabela.
    • <th> (Table Header): Define uma célula de cabeçalho. O texto geralmente é em negrito e centralizado por padrão.
  • <tbody>: Grupo de corpo da tabela. Contém os dados principais da tabela.
    • <tr> (Table Row): Define uma linha de dados.
    • <td> (Table Data): Define uma célula de dados.
  • <tfoot>: Grupo de rodapé da tabela. Contém resumos ou totais.
    • `colspan`: Atributo para <td> ou <th> que faz com que uma célula ocupe várias colunas. Ex: <td colspan="3">.
    • `rowspan`: Atributo para <td> ou <th> que faz com que uma célula ocupe várias linhas.
  • <caption> (Opcional): Define um título para a tabela. Deve ser o primeiro elemento dentro de <table>. 
    
<table>
    <caption>Vendas por Mês (2024)</caption>
    <!-- Resto da tabela -->
</table>
