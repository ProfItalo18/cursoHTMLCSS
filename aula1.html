Aula 1: HTML Básico - O Esqueleto da Web

Bem-vindo(a) à primeira aula do nosso curso de HTML e CSS! Aqui você dará os primeiros passos no mundo do desenvolvimento web, aprendendo a construir a estrutura fundamental de qualquer página da internet.

Prepare-se para entender como o HTML organiza o conteúdo, desde títulos até parágrafos, usando tags simples e poderosas.

O Que é HTML?

HTML (HyperText Markup Language) não é uma linguagem de programação, mas sim uma linguagem de marcação. Pense nele como o arquiteto que desenha a planta de uma casa: ele define onde estarão as paredes, as janelas, as portas – a estrutura principal do conteúdo que será exibido no navegador.

O HTML usa tags para "marcar" e descrever diferentes partes do seu conteúdo. Cada tag tem um propósito e ajuda o navegador a entender como exibir aquela informação.

Anatomia de uma Tag HTML

A maioria das tags HTML funciona em pares: uma tag de abertura e uma tag de fechamento. O conteúdo que você quer marcar fica entre elas.

<p>Este é um parágrafo de texto.</p>
  • <p>: É a tag de abertura.
  • </p>: É a tag de fechamento (sempre com uma barra /).
  • Este é um parágrafo de texto.: É o conteúdo do elemento.

Algumas tags são "vazias" ou auto-fecháveis, pois não precisam de conteúdo entre elas. Elas realizam uma ação específica.

<img src="imagem.jpg" alt="Descrição da imagem">
<br>
<hr>

Estrutura Essencial de um Documento HTML

Todo documento HTML segue uma estrutura básica que informa ao navegador como interpretar a página.


<!DOCTYPE html>
<html lang="pt-br">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Título da Aba do Navegador</title>
</head>
<body>
    <!-- Seu conteúdo visível vai aqui! -->
    <h1>Olá, Mundo!</h1>
    <p>Esta é a sua primeira página web.</p>
</body>
</html>

Entendendo Cada Parte:

  • <!DOCTYPE html>: Declaração que informa ao navegador que estamos usando a versão mais recente do HTML (HTML5).
  • <html lang="pt-br">: A tag raiz que engloba todo o conteúdo da página. O atributo lang="pt-br" indica o idioma principal.
  • <head>: Contém metadados sobre a página, como a codificação de caracteres (<meta charset="UTF-8">), configurações de responsividade (<meta name="viewport">) e o título que aparece na aba do navegador (<title>). **O conteúdo do <head> não é visível na página.**
  • <body>: Contém TODO o conteúdo visível da sua página web: textos, imagens, vídeos, links, etc.

Comentários HTML: Você pode adicionar comentários ao seu código usando <!-- seu comentário aqui -->. Eles são ignorados pelo navegador e servem para você e outros desenvolvedores entenderem o código.

Tags de Conteúdo Essenciais:

Cabeçalhos (<h1> a <h6>)

Usados para definir títulos e subtítulos, organizando a hierarquia do conteúdo. <h1> é o mais importante e deve ser usado apenas uma vez por página para o título principal.


<h1>Título Principal do Artigo</h1>
<h2>Seção Importante</h2>
<h3>Subseção</h3>
<h4>Detalhe Menor</h4;>

Parágrafos (<p>)

Usado para definir blocos de texto. Cada tag <p> cria um novo parágrafo.


<p>Este é o primeiro parágrafo do meu texto. Ele contém algumas informações importantes sobre o assunto.</p>
<p>Este é o segundo parágrafo. Note como o navegador adiciona um espaço entre eles automaticamente.</p>

Quebra de Linha (<br>)

Tag vazia que força uma quebra de linha, como um "Enter" dentro de um texto.


<p>Meu endereço é:
Rua dos Desenvolvedores, 123
Cidade do Código, BR</p>

Linha Horizontal (<hr>)

Tag vazia que cria uma linha horizontal, ideal para separar visualmente seções de conteúdo.


<p>Conteúdo da seção acima.</p>
<hr>
<p>Conteúdo da seção abaixo.</p>

Desafio Prático: Construa sua Primeira Página!

É hora de colocar a mão na massa! Crie um arquivo HTML e aplique o que aprendeu.

  1. Crie um novo arquivo chamado meu-perfil.html.
  2. Adicione a **estrutura básica completa** do HTML (!DOCTYPE, html, head, body).
  3. No <head>, defina o <title> como "Meu Perfil Pessoal - [Seu Nome]".
  4. Dentro do <body>, adicione:
    • Um <h1> com o seu nome.
    • Um <p> falando brevemente sobre você (seus hobbies, o que você gosta de fazer).
    • Um <hr> para separar.
    • Um <h2> com o título "Minhas Qualidades".
    • Três <p> separados, cada um descrevendo uma qualidade sua. Use <br> em um deles para quebrar uma linha.
  5. Salve o arquivo e abra-o no seu navegador para ver o resultado!

Você pode usar a seção "Sobre Mim" do seu `index.html` como inspiração, mas tente criar algo novo para este exercício.
